#  Especificação casos de uso
## Front End

#### Objetivo:
Gerar uma interface com 3 editores de texto, modelo, propriedades e variáveis, com realce de sintaxe e opção de fazer download dos arquivos criados, assim como visualização do resultado da execução do modelo.

#### Requisitos:
_nenhum_

#### Atores:
Usuário.

#### Pré Condições:
Acesso a URL do webpage.

#### Pós Condições:
Exibição do resultado da execução do modelo.

#### Fluxo Principal:
1. O sistema apresenta a tela principal ao usuário.
2. O usuário cria o modelo no editor de texto de modelo.
3. O usuário configura as propriedades do modelo no editor de texto de propriedades.
4. O usuário configura as variáveis de execução do modelo.
5. O usuário envia os arquivos ao servidor que executa o modelo e gera seus retornos.
6. O usuário verifica os retornos do modelo.
